# Django Commands

Initiate a project:

```sh
django-admin startproject [projectname]
```

Run project:

```sh
python manage.py runserver [ip:port]
```

Create migration:

```sh
python manage.py makemigrations
```

Apply migrations:

```sh
python manage.py migrate
```

Create super user:

```sh
python manage.py createsuperuser
```

Initiate an app:

```sh
python manage.py startapp [appname]
```
